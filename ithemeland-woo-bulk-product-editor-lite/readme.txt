== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/ithemeland-woo-bulk-product-editor-lite` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  After installation, a new "iT Woo Product" menu will be added to your admin dashboard.

== Frequently Asked Questions ==

= How is this different from the built-in WooCommerce editor? =
The default WooCommerce editor is very basic, requiring you to edit products one at a time for most fields. Our plugin provides a powerful spreadsheet-like interface and a bulk edit form, allowing you to update thousands of products and variations at once, saving you countless hours.

= Can I edit custom fields created by other plugins? =
Yes! The Pro version has full support for editing custom fields, including those created by the Advanced Custom Fields (ACF) plugin, allowing you to manage all your product data from one screen.

= What happens if I make a mistake? =
We've got you covered. The Pro version includes a complete history of all bulk edit operations. You can review every change and undo any action with a single click, giving you total peace of mind.

= Can I see a live demo? =
Absolutely. You can try the full power of the Pro version on our <a href="https://demos.ithemelandco.com/woocommerce-bulk-product-editing/">live demo site</a>.
